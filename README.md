Raytouch Renewables – Solar Energy Website

>Project Overview
This project is a responsive multi-page website developed for **Raytouch Renewables**, a solar energy solutions provider. It showcases residential, commercial, and industrial solar services and helps users understand the benefits of clean energy.

> Features
- Responsive design (Mobile, Tablet, Desktop)
- Home page with company overview
- Residential, Commercial & Industrial pages
- Portfolio section
- Contact form (Web3Forms API)
- SEO optimized (meta tags, sitemap, robots.txt)
-Modern UI with animations

> Technologies Used
- HTML5
- CSS3
- Bootstrap 5
-JavaScript

 >Project Structure
  >>>project-folder/
   ├── index.html
   ├── residential.html
   ├── commercial.html
   ├── industrial.html
   ├── assets/

* `index.html` – Main homepage
* `residential.html` – Residential page
* `commercial.html` – Commercial page
* `industrial.html` – Industrial page
* `privacy-policy.html` – Privacy policy
* `thank-you.html` – Form success page
* `assets/` – CSS, JS, images
* `robots.txt`, `sitemap.xml` – SEO

---

> How to Run

1. Download or clone the repository
2. Open folder in VS Code
3. Run `index.html` using Live Server

 >Contact Form
This project uses **Web3Forms API** to handle form submissions without backend.
  
  >Note
This is a frontend-only project. No backend or database is used.
